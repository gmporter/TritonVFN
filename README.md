TritonVFN
=========

An easy-to-use wrapper library around the Myricom Kernel-bypass (SNF) functionality

Dependencies:
 - cmake
 - libgtest-dev
 - libgflags-dev

To build the code:
  * $ mkdir build
  * $ cd build
  * build $ cmake ../src
  * build $ make -j
  * $ applications/my_app/my_app --help
